HTML - The Head Element

The HTML element is a container for the following elements: title, style, meta, link, script and base.

The head element is a container for metadata (data about data) and is placed between

HTML metadata is data about the HTML document. Metadata is not displayed on the page.

Metadata typically define the document title, character set, styles, scripts, and other meta information.

  1. The HTML title Element

    The title element defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.

    The content of a page title is very important for search engine optimization (SEO)! The page title is used by search engine algorithms to decide the order when listing pages in search results.

  3. The HTML link Element

    The link element defines the relationship between the current document and an external resource.

  4. The HTML meta Element

    The meta element is typically used to specify the character set, page description, keywords, author of the document, and viewport settings.

    Setting The Viewport

    The viewport is the user's visible area of a web page. It varies with the device - it will be smaller on a mobile phone than on a computer screen

  5. The HTML script Element

    The script element is used to define client-side JavaScripts.

  6. The HTML base Element

    The base element specifies the base URL and/or target for all relative URLs in a page
